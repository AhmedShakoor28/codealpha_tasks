# Credit Scoring Model

## Description
This project predicts whether a customer will default on a loan using machine learning techniques.

## Algorithms Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Tools & Libraries

- Pandas
- NumPy
- Scikit-learn
- Matplotlib
